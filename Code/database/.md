下面是sql建表语句


create database chat_db;

create table users(
    user_id varchar(15) not null,
    password varchar(20) not null,
    date datetime default current_timestamp,
    primary key(user_id)
);

CREATE TABLE msgs (
    rowadd int not null auto_increment,          
    sender_id VARCHAR(15) not null,
    receiver_id VARCHAR(15) not null,
    content VARCHAR(2000) not null,
    date datetime default current_timestamp,
    PRIMARY KEY (rowadd),                        
    INDEX idx_sender_receiver (sender_id, receiver_id)            
);



redis存储：
  账号密码： 哈希  总key : users , key:user_id, value:password
  消息缓存： List  总key ：msgs:sender_id:receiver_id (sender 和 receiver 排序后的结果) ,  value: sender_id, content, date  --json
                                                                                            //这里content是双重json，第一重解开具体内容json格式，第二重是客户端用来解开内容的json格式
